ONIVOX est/sera un réseau social basé sur l'audio. (www.onivox.com)
Mission
Vous offrir le meilleur...rien que cela !

Description
ONIVOX est/sera un réseau social basé sur l'audio ou chaque membre
pourra communiquer avec les autres non seulement par voie écrite
(pour ceux qui ne possèderaient pas le matériel adéquat) mais aussi
audio à travers différentes fonctionnalités et applications.
